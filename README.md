# CurseOfVanishing

A minecraft plugin for vanishing from other players

## Installation

[**Java 17**](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html) required

Download the JAR file from the [latest release](https://github.com/XenKys/CurseOfVanishing/releases/tag/1.0)

## Configuration file example

```yaml
fakeJoinMessage: "§aJoin > {player}"
fakeLeftMessage: "§cQuit > {player}"
inVanishJoinMessage: "§a{player} joined the game in vanish"
inVanishLeftMessage: "§a{player} left the game in vanish"
inVanishDeadMessage: "§a{player} dead in vanish"
```
